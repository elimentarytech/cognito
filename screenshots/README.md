# Screenshots

This directory contains screenshots for the Cognito AI extension.

## Required Screenshots

Please add the following screenshot files:

1. **onboarding-modal.png** - First-time onboarding experience showing the email and role selection modal
2. **cognito-sidebar.png** - Cognito AI sidebar with comments, JIRA ticket integration, and AI insights
3. **ai-configuration.png** - AI Configuration modal showing provider selection (OpenAI, Anthropic, Google Gemini)
4. **settings-integrations.png** - Settings & Integrations modal showing database, JIRA, and AI configuration options

## Adding Screenshots

1. Take screenshots of the extension in action
2. Save them with the exact filenames listed above
3. Place them in this `screenshots/` directory
4. Commit and push to the repository

The screenshots will be automatically displayed in the main README.md file.
