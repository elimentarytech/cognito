## üì• Quick Installation

### **Step 1: Download the Extension**
- Download the `cognito-ai-extension-protected.zip` file
- Save it to your computer (e.g., Desktop or Downloads folder)

### **Step 2: Extract the Files**
- Right-click on the `.zip` file
- Select "Extract All" or "Extract Here"
- This will create a folder with the extension files

### **Step 3: Install in Chrome**
1. Open Google Chrome
2. Go to `chrome://extensions/` (paste this in the address bar)
3. Enable "Developer mode" (toggle switch in the top-right corner)
4. Click "Load unpacked" button
5. Select the extracted folder
6. Click "Select Folder"

### **Step 4: Verify Installation**
- Look for the Cognito AI icon in your browser toolbar
- Click the icon to open the extension
- You should see the Cognito AI interface
